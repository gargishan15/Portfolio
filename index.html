Eventify

Developed a Responsive Event Management System with real-time scheduling, booking, and RSVP features, accessible across all devices. It enhances user engagement with countdown timers, optimized performance, and improved usability for seamless event experiences.

Sahara is a Doctor Appointment Booking System with 3-tier authentication (Patient, Doctor, Admin) that enables secure scheduling, payments, and profile management. It streamlines healthcare operations by improving booking efficiency, session management, and overall user experience.

Developed a frontend food and restaurant website showcasing dishes and restaurant details with responsive design and smooth navigation for enhanced user experience.

This project is a backend-driven chocolate website where users can securely sign up and log in to access features. Users can also share, update, and manage reviews, making the platform fully interactive and dynamic.

This project is a React-based weather application that fetches real-time weather data using a public Weather API. Users can search for any place to view current conditions such as temperature, humidity, and overall climate. The app provides a clean and responsive interface for quick and easy weather updates.
